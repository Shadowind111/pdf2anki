# Security Policy

## Sensitive Data

`pdf2anki` may process private PDFs and API keys. Never upload:

- `pdf2anki_config.json`
- `pdf2anki_state.db`
- private PDFs
- generated Anki decks containing copyrighted or private material

The example config file is safe to commit because it contains placeholders only.

## Reporting

Please open a GitHub issue for non-sensitive bugs. For sensitive issues, contact the repository maintainer privately.
